# Une Invitation à Paris 🗼

Elegantiškas prancūziško stiliaus kvietimas į Paryžių jubiliejaus proga.

## Paleidimas lokaliai

```bash
npm install
npm run dev
```

Atidaryti: http://localhost:3000

## Deploy į Vercel

1. Push šį projektą į GitHub repozitoriją
2. Eiti į [vercel.com](https://vercel.com) → "Add New Project"
3. Importuoti GitHub repozitoriją
4. Vercel automatiškai aptiks Next.js — tiesiog spausti **Deploy**

Arba naudoti Vercel CLI:

```bash
npm i -g vercel
vercel
```
